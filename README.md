# Java JDBC Connectivity

This repository demonstrates a structured implementation of JDBC in Java  
for establishing and validating a MySQL database connection.


### Overview

A clean and minimal reference for integrating Java applications with MySQL using the JDBC API.


### Features

• Standard JDBC connection pattern  
• MySQL connectivity example  
• Reusable boilerplate for projects & testing


### Setup

Update the database URL, username, and password in the source file.  
Compile and run to verify successful connection initialization.


### Purpose

Provide a concise and professional starting point for database integration  
in Java-based systems and development workflows.
